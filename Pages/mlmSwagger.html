MLM Network & Commission API

Version: 1.0.0

API for Multilevel & Unilevel commissions, Binary pairing, SKU-based commission configuration, member stores, inventory transfers, and wallet management.

Servers

  • Production: https://api.example.com
  • Sandbox: https://sandbox.api.example.com

Security: All endpoints require **Bearer Token (JWT)** authentication.

API Endpoints (Paths)

Member Types

GET /api/member-types

Get all member types

Response: 200 (List of MemberType)

POST /api/member-types

Create a member type

Request Body: MemberType

Response: 201 (Created MemberType)

GET /api/member-types/{id}

Get member type by id

Path Parameter: id (integer, required)

Response: 200 (MemberType)

PUT /api/member-types/{id}

Update member type

Path Parameter: id (integer, required)

Request Body: MemberType

Response: 200 (Updated)

DELETE /api/member-types/{id}

Delete member type

Path Parameter: id (integer, required)

Response: 204 (Deleted)
